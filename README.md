Task Manager — MERN Stack

A full-stack task manager application built with MongoDB, Express, React, and Node.js.

What I Practiced

1/Separation of concerns — frontend and backend are fully independent
2/REST API design with Express and organized into routes, controllers, and models
3/Connecting a React frontend to an Express backend using Vite's proxy
4/MongoDB data modeling with Mongoose
5/Environment variables to keep sensitive data out of the codebase

Tech Stack
Frontend: React, Vite, Material UI
Backend: Node.js, Express
Database: MongoDB Atlas, Mongoose
